#Font Family

In NoCodeWeb framework we have two defaults fonts. These fonts are : Roboto & Sans-Serif. By default we don't load Roboto font because it take long time. And if you don't need this font we load this for nothing. But you can eisely add it. To change the font of a text you can just add "ncw-font-f-roboto" or "ncw-font-f-sans-serif".

#Font Size

To change the size of your font it's pretty easy. You can add the class "ncw-font-s-" and the size you choose.
Sizes available: 6, 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 30, 36, 48, 60, 72.

#Font Weight

To change the font weight you can add the class "ncw-font-fw-" and the weight you want.
Weight available: 100, 300, 400, 500, 700, 900. (It can depend from your font)

#Line Height

To add a line height to your font you can just add the class "ncw-font-lh-" and the size you want.
Sizes available: 6, 8, 9, 10, 11, 12, 14, 16, 18, 20, 22, 24, 30, 36, 48, 60, 72, 86, 92, 106.

#Text Transform

To transform your text you can just add the class "ncw-font-tt-" and the transform you want to apply.
Transform available : none, lowercase, uppercase, capitalize.

Example

Text transform none

Text transform lowercase

Text transform uppercase

Text transform capitalize

        <p class="ncw-font-tt-none ncw-font-f-roboto">Text transform none</p>
<p class="ncw-font-tt-lowercase ncw-font-f-roboto">Text transform lowercase</p>
<p class="ncw-font-tt-uppercase ncw-font-f-roboto">Text transform uppercase</p>
<p class="ncw-font-tt-capitalize ncw-font-f-roboto">Text transform capitalize</p>
