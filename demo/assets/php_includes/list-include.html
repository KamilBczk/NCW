#Ul list

The ul element represents an unordered list of items, typically rendered as a bulleted list.

Example

  • First element
  • Second element
        <ul>
  <li>First element</li>
  <li>Second element</li>
</ul>

#List style

You can change the position of the 'marker' to inside or outside with two classes : "ncw-list-ls-inside" & "ncw-list-ls-outside".

Example

Inside:


  • First element
  • Second element

Outside:


  • First element
  • Second element
        <ul class="ncw-list-ls-inside">
  <li>First element</li>
  <li>Second element</li>
</ul>

<ul class="ncw-list-ls-outside">
  <li>First element</li>
  <li>Second element</li>
</ul>
