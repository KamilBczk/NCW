#Border (All)

To setup all border you can just add class "ncw-border-" and the size of border you want.
Sizes available : 1, 2, 5.

Example

Border 1px


Border 2px


Border 5px

        <p class="ncw-border-1">Border 1px</p>
<p class="ncw-border-2">Border 2px</p>
<p class="ncw-border-5">Border 5px</p>

#Border (X / Y)

To setup borders x (left & right) & y (top & bottom) you can add class "ncw-border-" "x-" or "y-" and size.
Size available: 1, 2, 5.

Example

Border x 1px


Border y 1px


Border x 2px


Border y 2px


Border x 5px


Border y 5px

        <p class="ncw-border-x-1">Border x 1px</p>
<p class="ncw-border-y-1">Border y 1px</p>
<p class="ncw-border-x-2">Border x 2px</p>
<p class="ncw-border-y-2">Border y 2px</p>
<p class="ncw-border-x-5">Border x 5px</p>
<p class="ncw-border-y-5">Border y 5px</p>

#Border (Color)

To change color of your border you can add class "ncw-border-color-" and the color you want.

Example

Orange color


Blue color


Green color

        <p class="ncw-border-1 ncw-border-color-orange">Orange color</p>
<p class="ncw-border-2 ncw-border-color-blue">Blue color</p>
<p class="ncw-border-5 ncw-border-color-green-dark">Green color</p>
