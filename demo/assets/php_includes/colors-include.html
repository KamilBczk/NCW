#Background colors

For adding the background-color to an element you can simply add a class on the element. The name of the class must beggin with "ncw-bg-" and after you add color from the color section.

Examples

Text

<h1 class="ncw-bg-color-blue-light">Text</h1>
<div class="ncw-bg-color-dark ncw-pos-w-25pr ncw-pos-h-50"></div>

#Font colors

For adding the font color to an element you can simply add a class on the element. The name of the class must beggin with "ncw-" and after you add color from the color section.
Warning : you can't add the transparency colors because it's not possible with just css to add transparency to a font.

Examples

Example text

<h2 class="ncw-color-orange">Example text</h2>

Grey text

<p class="ncw-color-grey-dark">Grey text</p>
