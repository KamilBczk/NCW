#Width

You can set up the width of an element with class "ncw-size-w-" and the value.
Value available : 10, 20, 25, 50, 100, 150, 200, 250, 300, 400, 500, 600, 700, 800, 900, 1000.

Example

#Width (percentage)

You can set up the with (with %) of an element with class "ncw-size-w-", the value & "pr" at the end.
Value available : 10, 20, 25, 30, 40, 50, 60, 70, 75, 80, 90, 100.
