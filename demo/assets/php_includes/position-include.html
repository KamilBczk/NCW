#Position static

The element is positioned according to the normal flow of the document. To set this position you can add class "ncw-pos-static".

Example

        <div class="ncw-pos-static ncw-size-w-50 ncw-size-h-50 ncw-bg-color-orange"></div>
<div class="ncw-pos-static ncw-size-w-50 ncw-size-h-50 ncw-bg-color-blue"></div>

#Position relative

The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left. To set this position you can add class "ncw-pos-relative".

Example

        <div class="ncw-pos-relative ncw-size-w-50 ncw-size-h-50 ncw-bg-color-orange"></div>
<div class="ncw-pos-relative ncw-size-w-50 ncw-size-h-50 ncw-bg-color-blue"></div>

#Position absolute

The element is removed from the normal document flow, and no space is created for the element in the page layout. To set this position you can add class "ncw-pos-absolute".

Example

        <div class="ncw-pos-absolute ncw-size-w-50 ncw-size-h-50 ncw-bg-color-orange"></div>
<div class="ncw-pos-absolute ncw-size-w-50 ncw-size-h-50 ncw-bg-color-blue"></div>

#Position fixed

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to the initial containing block established by the viewport, except when one of its ancestors has a transform, perspective, or filter property set to something other than none. To set this position you can add class "ncw-pos-fixed".

#Position sticky

The element is positioned according to the normal flow of the document, and then offset relative to its nearest scrolling ancestor and containing block (nearest block-level ancestor), including table-related elements, based on the values of top, right, bottom, and left. To set this position you can add class "ncw-pos-sticky".
